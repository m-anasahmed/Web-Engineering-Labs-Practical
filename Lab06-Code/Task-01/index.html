Welcome Alert Message

What is an Alert?

An alert() is a JavaScript method that displays a dialog box with a message to the user. It's one of the most basic ways to communicate with users.

When the page loads, it automatically displays a welcome message. You can also trigger it again by clicking the button below.

JavaScript Code:

alert("Welcome to JavaScript Learning Platform!");

Key Points:

  • Blocking: Alert stops code execution until dismissed
  • Simple: Easy way to display messages
  • User Interaction: Requires user to click OK
  • Debugging: Useful for testing and debugging

For Loop & Console Output

Loops are essential programming constructs that repeat code blocks. This section demonstrates a for loop that displays the first 10 natural numbers in the browser console.

What is a For Loop?

A for loop repeats a block of code a specified number of times. It's perfect for iterating through sequences or performing repetitive tasks.

For Loop Syntax:

for (initialization; condition; increment) {
  // code to be executed
}

Our Example - Display Numbers 1 to 10:

for (let i = 1; i <= 10; i++) {
  console.log(`Number: ${i}`);
}
