Even or Odd Checker

A simple JavaScript program to determine if a number is even or odd

What is Even and Odd?

Even Numbers: Numbers that are perfectly divisible by 2 with no remainder. Examples: 2, 4, 6, 8, 10, 12, 14, etc.

Odd Numbers: Numbers that are not divisible by 2, leaving a remainder of 1. Examples: 1, 3, 5, 7, 9, 11, 13, etc.

How to Check: Use the modulo operator (%) to find the remainder when dividing by 2. If number % 2 === 0, it's EVEN. If number % 2 !== 0, it's ODD.

// Main function to check if a number is even or odd
function checkEvenOdd() {
  // Step 1: Get input from user using prompt()
  const number = prompt("Enter a number:");

  // Check if user entered something and it's a valid number
  if (number === null) {
    alert("‚ùå You cancelled the prompt!");
    return;
  }

  if (isNaN(number) || number === "") {
    alert("‚ö†Ô∏è Please enter a valid number!");
    return;
  }

  const num = parseInt(number);

  // Step 2: Check if the number is even or odd using if-else
  if (num % 2 === 0) {
    // If remainder is 0, it's even
    alert(`‚úÖ ${num} is an EVEN number`);
  } else {
    // If remainder is 1, it's odd
    alert(`‚úÖ ${num} is an ODD number`);
  }
}
