Features

Dynamic Header

Click the buttons above to change the header text dynamically using JavaScript. No page refresh needed!

Color Selection

Select from the dropdown or click color boxes to instantly change the background color of this content area.

Auto-Loading Footer

A special footer element is automatically added 5 seconds after the page loads using JavaScript setTimeout().

How It Works

JavaScript Features Implemented

  • Header Text Update: Uses DOM manipulation to change h1 and p text on button click
  • Background Color Change: Uses select dropdown and onclick handlers to modify CSS background-color property
  • Dynamic Footer: Uses setTimeout() to add a new element after 5 seconds
  • Status Messages: Shows feedback messages when actions are completed
  • Event Handling: Multiple event listeners for user interactions
