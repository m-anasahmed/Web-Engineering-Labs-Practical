Feature 1: Change Paragraph Content

How It Works

When you click the "Change Content" button, JavaScript updates the paragraph text using the textContent property. Each time you click, it cycles through different messages.

  • Event: click event on the button
  • Method: getElementById() to select the paragraph
  • Property: textContent to change the text
  • Feature: Cycling through multiple messages
// Array of different messages
const messages = [
  "Original message...",
  "Message 1: Great job clicking the button!",
  "Message 2: JavaScript is amazing!",
  "Message 3: DOM manipulation is powerful!"
];

// Function to change paragraph content
function changeParagraphContent() {
  const paragraph = document.getElementById('mainParagraph');
  messageIndex = (messageIndex + 1) % messages.length;
  paragraph.textContent = messages[messageIndex];
}

Feature 2: Button Style on Mouse Events

How It Works

The button changes its appearance when you hover over it using mouseover and mouseout events. This provides visual feedback to the user.

  • Event 1: mouseover - when mouse enters the button
  • Event 2: mouseout - when mouse leaves the button
  • Method: style property to modify CSS
  • Effect: Color change and scale animation
// Add mouseover event listener
button.addEventListener('mouseover', function() {
  // Change button appearance on hover
  this.style.background = 'linear-gradient(135deg, #ff6b6b 0%, #ff5252 100%)';
  this.style.transform = 'scale(1.05)';
  this.style.boxShadow = '0 8px 25px rgba(255, 107, 107, 0.4)';
});

// Add mouseout event listener
button.addEventListener('mouseout', function() {
  // Reset button appearance
  this.style.background = 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)';
  this.style.transform = 'scale(1)';
  this.style.boxShadow = '0 4px 15px rgba(102, 126, 234, 0.3)';
});

Feature 3: Dynamically Add Paragraphs

How It Works

Each time you click the button, a new paragraph element is created and added to the page dynamically using JavaScript DOM methods.

  • Method 1: createElement() - creates a new element
  • Method 2: appendChild() - adds element to the DOM
  • Method 3: setAttribute() - sets element attributes
  • Feature: Each paragraph gets a unique timestamp
// Function to add new paragraph dynamically
function addNewParagraph() {
  // Create a new div element
  const newPara = document.createElement('div');
  
  // Set the class name
  newPara.setAttribute('class', 'paragraph-item');
  
  // Set the content with HTML
  newPara.innerHTML = `New paragraph #${paragraphCount} at ${timestamp}`;
  
  // Add the element to the container
  document.getElementById('paragraphsContainer').appendChild(newPara);
}
